# lingmo-wiki
Official Wiki of Lingmo, powered by [Vitepress](https://vitepress.dev).

## Running locally
### pnpm
```shell
git clone https://github.com/LingmoOS/lingmo-wiki
cd lingmo-wiki
pnpm i
pnpm vitepress dev docs
```
### npm
```shell
git clone https://github.com/LingmoOS/lingmo-wiki
cd lingmo-wiki
npm install
npx vitepress dev docs
```
