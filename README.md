# lingmo-wiki
[Official Wiki of Lingmo](https://wiki.lingmo.org), powered by [Vitepress](https://vitepress.dev).

## Running locally
### pnpm
```shell
git clone https://github.com/LingmoOS/lingmo-wiki
cd lingmo-wiki
pnpm i
pnpx vitepress dev docs
```
### npm
```shell
git clone https://github.com/LingmoOS/lingmo-wiki
cd lingmo-wiki
npm install
npx vitepress dev docs
```
